## PHASE 1 — WhatsApp Retention Engine

**Goal**: Turn existing customers into repeat buyers. This is the highest ROI activity right now.

### The Database Asset
- Bill data = transaction history (date, amount, items if captured)
- Mobile numbers = direct WhatsApp access
- **VIP threshold**: Anyone who has bought 2+ times OR single bill ≥ ₹3,500

### Customer Segmentation (do this first)

| Segment | Criteria | Size estimate | Strategy |
|---------|----------|---------------|----------|
| **VIPs** | 2+ purchases OR ₹3,500+ single bill | Small but gold | White-glove, early access, personal message from Priya |
| **One-timers** | Bought once, 30–60 days ago | Medium | Win-back campaign, "aapke liye kuch naya aaya hai" |
| **Recent buyers** | Bought in last 2 weeks | Small | Nurture, don't sell yet — share content |
| **Cold** | Bought 60+ days ago, never returned | Largest | Re-engagement with a hook |

### WhatsApp Campaign Calendar (Monthly Rhythm)

**Week 1** — New arrivals broadcast (all segments except recent buyers)
**Week 2** — VIP early access (VIPs only — "pehle aap dekhein")
**Week 3** — Festival/occasion hook (if applicable) or style inspiration
**Week 4** — Win-back for cold customers ("hum yaad karte hain aapko")

### WhatsApp Message Principles
- Always personal tone — "Priya di ki taraf se" energy
- Never bulk-paste — use customer's first name if possible
- 1 clear CTA: visit store, WhatsApp to order, or link to product
- Never more than 2 messages per week to any one person
- Include a product image always (visual = open rate)
- Read references/whatsapp-templates.md for ready-to-use message copy

### VIP Customer Protocol
When Sid identifies a VIP:
1. Send a personal voice note or personal WhatsApp (not broadcast) from Priya
2. Offer: "aapke liye pehle dikhate hain naya collection"
3. Track: note in DB that VIP outreach was done
4. Invite to exclusive preview if doing an exhibition

## PHASE 2 — Ecommerce Activation

**Goal**: Get first 10 orders on labelethnicvogue.shop within 60 days.

### Why Zero Purchases Happen (and fixes)

| Root cause | Fix |
|-----------|-----|
| No trust signals | Add customer photos, reviews, WhatsApp chat button |
| No urgency | Show "X pieces left", limited stock callout |
| No traffic | WhatsApp campaigns with direct product links |
| Checkout friction | Test the full checkout flow yourself — fix any errors |
| No payment options | Ensure UPI, cards, CoD all work |

### Ecommerce Activation Sequence (do in order)

**Week 1 — Trust Setup**
- [ ] Add 3–5 real customer photos to product pages (get permission)
- [ ] Add WhatsApp chat button (use WA.me link with pre-filled message)
- [ ] Write proper product descriptions (use ethnic-vogue-copywriter skill)
- [ ] Add "Free shipping above ₹2,000" prominently

**Week 2 — Traffic Push**
- [ ] Send WhatsApp broadcast to ALL captured numbers: "Ghar baithe order kar sakti hain ab — labelethnicvogue.shop"
- [ ] Post 3 Instagram Stories with direct link sticker to website
- [ ] Instagram Reel featuring 3 products with "link in bio" CTA

**Week 3 — Conversion Fix**
- [ ] If visitors but no orders: check Google Analytics / WooCommerce traffic
- [ ] If no visitors: repeat traffic push, do a Story poll ("online order karogi?")
- [ ] Add a WhatsApp order option as fallback ("prefer karo toh WhatsApp pe bhi le lo")

**Week 4 — First Order Celebration**
- [ ] When first online order comes: send a personal thank-you WhatsApp + small surprise (a handwritten note in parcel works magic)
- [ ] Screenshot and post (with permission) — social proof flywheel begins

### Ecommerce Content That Converts
- Video of dupatta drape / fabric texture (trust in fabric quality)
- "How to order" Story highlight on Instagram
- Size guide clearly on every product page

## PHASE 3 — Performance Marketing Foundation

**Goal**: Build the data infrastructure for paid ads before spending a rupee.

### Before Running Any Paid Ad, Verify:
- [ ] Facebook Pixel installed on WooCommerce site (via Meta Pixel plugin)
- [ ] Standard events firing: ViewContent, AddToCart, Purchase
- [ ] WhatsApp Business API connected (or at least WhatsApp Business app)
- [ ] Custom Audiences created in Meta: website visitors, customer list upload
- [ ] Instagram profile has 500+ followers and consistent posting for 30+ days

### Meta Pixel Setup Priority
This is the most important technical task in Phase 3. Without pixel data, retargeting is impossible.

Steps:
1. Install "PixelYourSite" plugin on WooCommerce
2. Connect to Meta Business Suite with Pixel ID
3. Verify events in Meta Events Manager
4. Let it cook for 30 days minimum before running ads

### Audience Building (before ads)
- Upload customer mobile numbers to Meta as Custom Audience
- Create Lookalike Audience from that list (1% India lookalike = gold)
- Create website visitor audience (needs pixel running for 30 days)

## PHASE 4 — Paid Ads & Scale

**Read references/paid-ads-playbook.md for full details.**

### Ad Types in Priority Order

1. **WhatsApp Click-to-Chat ads** — cheapest, highest conversion for ethnic wear
2. **Retargeting — website visitors** — needs pixel data from Phase 3
3. **Cart abandonment** — WooCommerce + Meta integration
4. **Cold audience — Lookalike** — only after 50+ online purchases
5. **Instagram Reels boost** — boost organic content that already performed well

### The "Woh Wali Feeling" Retargeting Framework
(Existing customer re-engagement via Meta Ads)

- **Week 1 (Awareness)**: "Naya aaya hai" content — show new arrivals to past customers
- **Week 2 (Consideration)**: Specific product they may have seen — fabric detail video
- **Week 3 (Conversion)**: Direct offer — free shipping, limited stock, or VIP note

## DAILY ACTIONABLES — By Phase

### Right Now (Phase 1+2 Focus)

**Daily (5 minutes)**
- Post 1 Instagram Story (product/behind-the-scenes/fabric detail)
- Reply to all DMs and WhatsApp messages within 2 hours

**Weekly (1 hour)**
- Monday: Plan week's content — 3 Stories, 1 Reel, 1 WhatsApp broadcast

- Friday: Pull week's numbers — how many new numbers captured, any website visits, any online orders?

**Monthly (2 hours)**
- Update customer database — tag VIPs, segment by recency
- Send VIP personal outreach
- Review: what content got most saves/shares? Double down on that.
- Plan next exhibition strategy

### Exhibition Strategy Add-on
Exhibitions are the primary acquisition channel right now. Maximize them:
- **Capture rate goal**: Push from 50% to 80% mobile number capture
  - Tactic: "Hamare WhatsApp updates lene ke liye number dijiye — exhibition special pieces pehle aapko dikhenge"
- **Post-exhibition WhatsApp**: Within 48 hours, message everyone who bought: "Thank you, aapke saath achha laga" + one new arrival teaser
- **Non-buyers**: Message people who enquired but didn't buy: "Woh piece abhi bhi available hai" or "Naya aaya hai aapke liye"
