## Typography System

### Font Role Hierarchy (use all four in combination)

| Role | Font Family | Usage |
|------|-------------|-------|
| **Display / Hero Title** | Playfair Display, Cormorant Garamond, or similar editorial serif | Main headline — largest, most dominant element on the canvas |
| **Body / Details** | Merriweather | Event details, descriptions, dates, addresses — readable, warm serif |
| **Accent / Emotion** | Any handwritten/script font (e.g., Dancing Script, Pacifico, Satisfy) | Short poetic phrases, taglines, "before the season fades…" — adds soul |
| **Practical Info** | Any monospaced font (e.g., Courier Prime, Space Mono) | Dates in numeric form, venue codes, structured data — creates editorial contrast |

### Typography Rules
- **Title dominates.** The main event/product name must be the largest element — visually unmistakable
- **Mix all four font roles** in every design — the contrast between serif + handwritten + mono is the brand signature
- **Hierarchical sizing**: Display title > Merriweather body > Handwritten accent ≈ Monospaced info
- **Title case** for display headings always
- **Script accents stay short** — 3–6 words max, poetic, evocative
- **Monospaced block** for all dates/times/venue — treat it like a label or stamp

## Color Palette

### Primary Brand Colors
| Name | Hex | Use |
|------|-----|-----|
| Indigo / Deep Navy | `#1a2940` | Primary text, dark backgrounds, headings |
| Rust / Terracotta | `#c4793c` | Accent color, decorative rules, subheadings |
| Cream / Ivory | `#fdf6ed` | Primary background — the "paper" |
| Warm Gold | `#c9a84c` | Highlight details, borders, ornamental lines |
| Rama Green | `#3d6b5e` | Occasional accent — earthy, grounding |

### Earthy Neutral Palette (Sid's preference)
- Backgrounds: cream, warm ivory, off-white, aged parchment tones
- Text: deep indigo, dark espresso brown — **never pure black (#000000)**
- Accents: terracotta, rust, muted gold — warm, not neon
- **Avoid**: Cool greys, bright whites, electric colors, any neon

### Color Combination Rules
- **Background**: Always cream/ivory — never white, never dark
- **Primary text**: Indigo (`#1a2940`) on cream
- **Accent lines / rules**: Rust or gold
- **Never use more than 3–4 colors** in a single design

---

## Layout Principles

### Composition
- **Typography-first**: Let type carry the design — no garment photos needed
- **Generous whitespace**: Earthy, gallery-like breathing room
- **Centered or left-aligned** editorial layouts — avoid asymmetric chaos
- **Thin decorative rules** (horizontal lines) between sections — a hallmark of the style
- **Stamp/label elements** for practical info blocks (venue, dates) — boxed or ruled off

### Mood Keywords
*editorial, warm, artisanal, summer farewell, gallery invitation, craft heritage, unhurried luxury*

## Canva Prompt Formula

When generating any Canva design for Ethnic Vogue, always include:

```
Typography-only design (no garment photos).
Font stack:
- Display serif (Playfair Display / Cormorant Garamond) for the main title — dominant
- Merriweather for body details
- Handwritten script for 1–2 accent phrases
- Monospaced for dates/venue block

Color palette: Indigo (#1a2940), Rust (#c4793c), Cream/Ivory (#fdf6ed), Warm Gold (#c9a84c)
Background: Cream/ivory textured — never white
Mood: Editorial, earthy, artisanal, warm — Indian craft heritage meets luxury gallery invite
```

## Common Asset Templates

### Exhibition / Event Invite (Portrait Poster)
- **Hero**: Event/collection name in display serif — massive
- **Accent**: Handwritten tagline below ("before the season ends…")
- **Details block**: Dates in monospaced, venue in Merriweather
- **Brand name**: "Ethnic Vogue by Priya" in small caps or light Merriweather at bottom

### Instagram Story / Square Post
- Same font hierarchy, condensed
- Display title takes 50–60% of canvas height
- Brand name as a small footer

### WhatsApp / Flyer (Landscape)
- Display title left-aligned, large
- Monospaced date/venue block right side
- Thin rust rule dividing sections

## What to Avoid
- ❌ Garment / product photos (unless explicitly requested)
- ❌ Pure white backgrounds
- ❌ Cool-toned greys or blues
- ❌ Sans-serif-only layouts (always mix with serif)
- ❌ Bright or neon accent colors
- ❌ Cluttered layouts — less is always more
- ❌ Generic "Indian" stock art (elephants, lotus clip art, etc.)
